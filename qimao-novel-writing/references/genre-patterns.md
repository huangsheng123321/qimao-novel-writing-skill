# 网文流派套路与爽点设计

## 玄幻/修仙

### 力量体系设计模板

```text
境界划分建议 9-12 个大境界，每个分初/中/后/圆满：
练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 真仙 → 金仙 → 大罗
```

每个境界要有标志性能力。例如：

- 金丹：寿元大增，可御剑飞行。
- 元婴：元婴出窍，等于第二条命。
- 化神：神识覆盖百里。
- 渡劫：引动天劫，天地异象。

经典爽点：

1. 测试天赋：众人嘲笑 → 测试石爆碎 → 全场震惊。
2. 越级挑战：高境界碾压 → 主角反杀 → 打破常识。
3. 秘境夺宝：众人争抢 → 主角暗中得手 → 被发现 → 反杀夺宝。
4. 宗门大比：被轻视 → 一路碾压 → 决赛惊艳 → 长老争抢收徒。
5. 退婚流：上门退婚 → 主角放话 → 快速崛起 → 对方后悔。
6. 身份揭露：被当废物 → 实际是传人/转世 → 身份揭露 → 众人跪舔。

## 都市/战神

身份层级：

```text
普通富豪 < 地方豪强 < 省级世家 < 京城豪门 < 隐世家族 < 主角
```

经典场景：

1. 宴会打脸：反派炫耀 → 主角身份揭露 → 大佬跪舔 → 反派崩溃。
2. 商场护妻：妻子被欺负 → 主角出现 → 一个电话 → 对方破产。
3. 医术救人：众人束手无策 → 主角出手 → 起死回生 → 权贵欠人情。
4. 地下拳场：被嘲笑 → 一拳 KO → 庄家赔惨 → 收服势力。
5. 同学聚会：被攀比嘲讽 → 主角低调展示实力 → 全场沉默。

## 言情/甜宠

男主类型：

| 类型 | 特点 | 台词风格 |
| --- | --- | --- |
| 霸道总裁 | 强势、占有欲强、护短 | “我的人，谁动一下试试。” |
| 高冷禁欲 | 表面冷淡，只对女主温柔 | “别闹。” |
| 病娇偏执 | 极度依赖，不择手段 | “你只能是我的。” |
| 温柔竹马 | 默默守护，永远兜底 | “我一直在。” |
| 年下狼狗 | 直接、热烈、会撒娇 | “姐姐，看看我。” |

甜梗：误会解除、吃醋现场、身份反转、先婚后爱、重生追妻。

## 悬疑/灵异

氛围方式：规则怪谈、细思极恐、未知恐惧、线索反转。

推理结构：

```text
案件发生 → 表面线索指向A → 深入调查发现矛盾 → 线索重组指向B → 关键证据出现 → 真凶C
```

真相要“意料之外，情理之中”。

## 历史/穿越

穿越者优势：

1. 知道历史走向。
2. 现代商业知识。
3. 诗词装逼。
4. 美食征服。
5. 简单科技发明。
6. 现代医学常识。

禁忌：科技树不能跳太快；官职礼仪要基本合理；若改变大历史事件，最好设定为架空。
