# 00 爆款小说调研与结构分析

## 用户启动提示词

```text
请帮我查找番茄男频小说最近一个月的爆款小说（阅读量、完读量），分析该小说的整体框架是什么样子的，钩子是怎么设计的，高潮是怎么设计的等等。

然后给我 3-5 个仿写建议，并推荐一个，问我确认可以不。
```

## 任务

1. 查找/参考番茄男频最近一个月爆款小说。
2. 分析阅读量、完读量、完读率等数据可见性。
3. 拆解整体框架、黄金三章、钩子、高潮、爽点、人设、冲突升级。
4. 给 3-5 个仿写建议。
5. 推荐一个。
6. 结尾必须问：

```text
我建议优先选【方案X】。你确认可以不？
```

## 数据规则

不能编造阅读量/完读量。无法确认时说明：

```text
当前无法实时确认番茄后台真实阅读量/完读量，我会基于公开榜单、热度表现、评论反馈和同题材爆款结构做分析，不编造具体数据。
```
