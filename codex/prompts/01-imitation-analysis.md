# 仿写拆解与三案任务

当用户已经确认某个仿写建议，或用户直接提供爆款文、样章、参考文时，执行本任务。

请完成：

1. 主题方案
2. 框架方案
3. 仿写方案
4. 推荐采用哪一个，或如何组合

注意：

- 不要照搬原文设定、人物、冲突。
- 只能借节奏、爽点、钩子设计。
- 不要直接写正文。
- 必须等待用户确认三案方向。

输出格式：

```markdown
## 主题 / 框架 / 仿写三案

### A. 主题方案
- 核心卖点：
- 情绪价值：
- 适合读者：
- 平台适配：
- 风险：

### B. 框架方案
- 主线：
- 黄金三章：
- 第一卷结构：
- 长线钩子：
- 风险：

### C. 仿写方案
- 借鉴什么：
- 必须改掉什么：
- 新故事核：
- 爽点迁移方式：
- 风险：

## 我的建议
我建议采用【X】为主，【Y】为辅，原因：

请用户确认采用 A/B/C，或确认混合方案。
```
