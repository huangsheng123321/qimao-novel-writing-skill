# 更新记录

## v1.2.0

- 按用户最新要求固化状态机流程：爆款调研 → 3-5个仿写建议 → “确认”进入三案 → “确认”进入大纲人物细纲正文规划 → “确认”写5章 → “确定”审核修改最终稿。
- 新增默认启动提示词：查找番茄男频最近一个月爆款小说，分析阅读量、完读量、框架、钩子、高潮等。
- 明确用户回复“不行/重新推荐/重新给方案”时回退到对应阶段。
- 明确不得编造阅读量、完读量等不可见数据。

## v1.1.0

- 优化默认流程：用户先给主题时，先查找/参考同题材高热小说或爆款案例。
- 新增“主题调研与爆款分析”阶段。
- 新增 3-5 个仿写建议输出，并要求先等待用户确认。
- 用户确认仿写建议后，再输出主题方案、框架方案、仿写方案三案。
- 同步更新 Codex AGENTS.md 与 prompts/00-topic-research.md。

## v1.0.0

- 初版七猫/番茄网文写作技能。
- 支持 Hermes Agent Skill 格式。
- 支持 Codex CLI 项目模板格式。
- 支持仿写拆解：主题方案、框架方案、仿写方案。
- 支持整本大纲、人物角色、章节细纲。
- 支持五章一组：写作、审核、修改终稿。
- 增加黄金三章、爽点、钩子、去 AI 味规则。
